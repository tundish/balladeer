Looking for Balladeer?

Good. You have discovered the page of important links.

Balladeer is a Python framework. It helps you build interactive web-based multimedia stories.

  • There is an occasional Development Blog.
  • But if you want the tutorial you should start here.
  • The markup system is easy to learn. It's called SpeechMark.
  • TOML data files are used to define scenes.
  • The code is available from GitHub.
  • Of special interest will be the examples.
  • The API is fully documented online.
  • The Python package is hosted on PyPI.
  • IFWiki keeps a record of new releases.
  • If you need any help, you have only to ask.
